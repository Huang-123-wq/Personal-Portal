- **核心引擎层**：基于 Python 的 BM25 搜索算法（`scripts/search.py`, `core.py`）处理 CSV 知识库（`data/*.csv`），支持按领域（domain）和技术栈（stack）进行多维度检索。
- **技能定义层**：采用 `SKILL.md` 作为 AI Agent 的核心交互接口，定义了从需求分析到设计系统生成的标准化工作流（Step 1-4）。
- **分发安装层**：通过 TypeScript 编写的 CLI 工具（`cli/src/index.ts`）实现模板化生成，将技能动态注入到 Claude、Cursor、Qoder 等多种 AI 编程助手的本地环境中。
- **数据驱动架构**：所有设计规范（如 67 种 UI 风格、161 种配色方案）均存储在结构化 CSV 文件中，实现了内容与逻辑的解耦。
