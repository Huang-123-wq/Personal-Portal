## 项目简介

DataViz 是一个强大的数据可视化平台，帮助用户将复杂数据转化为直观的图表。支持折线图、柱状图、散点图、热力图等多种可视化类型。

## 核心功能

- **多种图表** — 支持 15+ 种图表类型
- **自定义仪表板** — 拖拽组合多个图表
- **实时数据** — 连接 API 实时更新
- **导出功能** — 支持 PNG、SVG、PDF 导出

## 技术架构

基于 React + D3.js 构建，使用 Vite 作为构建工具。图表渲染完全自定义，不依赖第三方图表库，保证了最大的灵活性。
