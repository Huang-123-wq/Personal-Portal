## 项目简介

TaskFlow 是一个极简主义任务管理工具，灵感来源于 Linear 的设计语言。它专注于速度和效率，让团队可以在几秒钟内创建、分配和追踪任务。

## 核心功能

- **看板视图** — 拖拽式任务管理
- **快捷键** — 全键盘操作支持
- **实时协作** — 基于 WebSocket 的实时更新
- **智能筛选** — 按标签、优先级、负责人筛选

## 技术亮点

使用 Next.js App Router 构建，数据库采用 PostgreSQL + Prisma ORM。前端使用 Tailwind CSS 实现暗色主题，与 Linear 设计系统保持一致。
